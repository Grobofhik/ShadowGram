# <img src="../../resources/icons/folder_icon.png" width="32" valign="middle"> Создание нового профиля

В этом разделе вы узнаете, как правильно добавить новый Telegram-аккаунт в систему ShadowGram для дальнейшей работы и автоматизации.

---

## 📝 Пошаговая инструкция

Для добавления аккаунта используйте специальную панель **"Добавить новый профиль"**, которая расположена в нижней части главного окна приложения.

### 1. Имя профиля
Введите любое удобное для вас название. Это имя будет отображаться в списке аккаунтов. Рекомендуется использовать понятные имена (например, `Acc_01`, `Market_Manager` и т.д.).

### 2. Выбор рабочей директории
Нажмите на иконку <img src="../../resources/icons/folder_icon.png" width="18" valign="middle"> справа от поля ввода пути. 
*   Выберите папку, в которой будет храниться **Tdata** и сессия вашего аккаунта (`.session` файл и другие данные).
*   **Важно:** Каждый аккаунт должен иметь свою уникальную папку!

### 3. Настройка прокси (Опционально)
В поле **HTTP Proxy** введите данные вашего прокси-сервера. 
*   **Формат:** `http://user:pass@host:port`
*   После ввода нажмите на кнопку <img src="../../resources/icons/new_proxy_icon.png" width="18" valign="middle"> для мгновенной проверки соединения. Если прокси рабочий, кнопка станет зеленой.

### 4. Завершение создания
Нажмите кнопку **"Создать"**. Новый профиль мгновенно появится в вашем списке.

---

## 💡 Советы
*   **Безопасность**: Всегда проверяйте прокси перед созданием профиля, чтобы избежать лишних запросов к серверам Telegram с вашего реального IP.
*   **Аватарка**: Если вы укажете путь к уже существующей папке с сессией, система подхватит её. Чтобы аватарка отобразилась в списке, запустите модуль **"Проверка сессии"**.
