# <img src="../../resources/icons/settings_icon.png" width="32" valign="middle"> Управление профилем

Каждая карточка аккаунта в ShadowGram — это полноценный пульт управления сессией. Ниже приведено детальное описание всех интерактивных элементов.

---

## ⚡ Основные действия

В правой части каждой карточки находится главная кнопка управления состоянием сессии:

| Действие | Иконка | Описание |
| :--- | :---: | :--- |
| **Запустить** | <img src="../../resources/icons/start_icon.png" width="20"> | Инициализирует запуск Telegram. Если для профиля настроен прокси, система автоматически поднимет туннель. |
| **Закрыть** | <img src="../../resources/icons/cancel_icon.png" width="20"> | Принудительно завершает процесс Telegram и корректно разрывает соединение с прокси. |

> **Статус сессии:** Текстовый индикатор рядом с кнопкой ("Запущен" / "Остановлен") меняет цвет в зависимости от реальной активности процесса.

---

## 🛠 Панель инструментов (Центр)

Эти кнопки позволяют настраивать и диагностировать аккаунт "на лету", не открывая само окно Telegram.

### 🔍 Диагностика и Безопасность
*   **<img src="../../resources/icons/search_icon.png" width="18" valign="middle"> Проверить сессию**: Выполняет скрытый запрос к серверам Telegram через MTProto. Проверяет валидность ключей, наличие бана и обновляет аватарку.
    *   `Зеленый` — сессия активна.
    *   `Красный` — сессия невалидна или аккаунт заблокирован.
*   **<img src="../../resources/icons/proxy_icon.png" width="18" valign="middle"> Проверить прокси**: Тестирует доступность привязанного прокси-сервера. Если прокси "упал", вы увидите уведомление об ошибке.
*   **<img src="../../resources/icons/succure_icon.png" width="18" valign="middle"> Privacy Guard**: (Доступно через модули) Мгновенно выставляет настройки скрытности (скрыть номер, запретить звонки).

### ⚙️ Конфигурация
*   **<img src="../../resources/icons/new_proxy_icon.png" width="18" valign="middle"> Изменить прокси**: Быстрая правка данных прокси-сервера (`http://user:pass@host:port`).
*   **<img src="../../resources/icons/device_icon.png" width="18" valign="middle"> Имя устройства**: Позволяет задать уникальный Hostname для сессии. Это имя будет видеть Telegram в списке активных устройств аккаунта.
*   **<img src="../../resources/icons/note_icon.png" width="18" valign="middle"> Заметки**: Редактирование текстового комментария к аккаунту. Чтобы быстро прочитать заметку, просто наведите курсор на эту иконку.

### 📁 Работа с файлами
*   **<img src="../../resources/icons/folder_icon.png" width="18" valign="middle"> Открыть папку**: Мгновенный переход в рабочую директорию профиля (workdir) через системный проводник.
*   **<img src="../../resources/icons/cash_icon.png" width="18" valign="middle"> Чистка кэша**: Безопасное удаление временных файлов (логов и картинок) Telegram, которые переполняют диск.
*   **<img src="../../resources/icons/delete_icon.png" width="18" valign="middle"> Удалить**: Убирает аккаунт из списка приложения. *Сами файлы на диске остаются нетронутыми.*

---

## ↕ Навигация и выбор

*   **Сортировка (▲ / ▼)**: Используйте стрелки в левой части карточки, чтобы выстроить аккаунты в нужном вам порядке. 
*   **Массовый выбор**: Отметьте чекбокс для включения аккаунта в список массовых операций (запуск, остановка или проверка прокси для группы сессий).

---

<img src="../../resources/icons/note_icon.png" width="24" valign="middle"> *Подсказка: Для максимально плавной прокрутки списка аккаунтов используйте колесико мыши. Если список большой, вы можете воспользоваться поиском в верхней части окна.*
