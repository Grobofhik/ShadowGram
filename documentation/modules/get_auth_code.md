# <img src="../../resources/icons/search_icon.png" width="32" valign="middle"> Get Auth Code (Получить код)

Модуль **Get Auth Code** предназначен для перехвата сервисных сообщений от официального аккаунта Telegram (ID: `777000`). Это необходимо, когда вам нужно войти в один из ваших аккаунтов на другом устройстве (например, в мобильном приложении или Telegram Desktop).

---

## 🔑 Как использовать модуль?

1.  **Запустите вход**: На новом устройстве введите номер телефона вашего аккаунта.
2.  **Ожидайте запрос**: Telegram отправит код подтверждения в "другое приложение" (вашу сессию в ShadowGram).
3.  **Запустите модуль**: Отметьте нужный аккаунт в списке ShadowGram и выберите модуль **Get Auth Code**.
4.  **Получите результат**: Бот проверит последние сообщения. Если код пришел, он будет выведен в логах крупным шрифтом.

---

## ⚙️ Режимы работы

Модуль работает в два этапа:
*   **Проверка истории**: Сначала бот смотрит последние 3 сообщения. Если вы запустили модуль через пару секунд после запроса кода, он, скорее всего, уже будет там.
*   **Ожидание (Live)**: Если в истории кода нет, бот переходит в режим ожидания (по умолчанию 60 секунд) и мониторит новые входящие сообщения от Telegram в реальном времени.

---

## ⚠️ Важные замечания

*   **Безопасность**: Модуль только читает сообщения и ничего не отправляет. Это абсолютно безопасно для сессии.
*   **Ограничения**: Не запрашивайте коды слишком часто на один и тот же аккаунт — это может привести к временному ограничению (Flood Wait) со стороны Telegram.
*   **Имя модуля**: В консоли вы увидите "Код подтверждения: XXXXX". Скопируйте эти 5 цифр и введите в поле входа на новом устройстве.
