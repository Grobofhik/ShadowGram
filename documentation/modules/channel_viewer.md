# <img src="../../resources/icons/view_icon.png" width="32" valign="middle"> Channel Viewer (Просмотры)

Модуль **Channel Viewer** предназначен для массовой накрутки просмотров на посты в Telegram-каналах. Это один из самых мощных инструментов в ShadowGram, поддерживающий полную параллельность и эмуляцию человеческого поведения.

---

## 🛠 Принцип работы

В отличие от простых скриптов, наш модуль имитирует реального пользователя:
1.  **Авторизация**: Бот заходит в аккаунт, используя сохраненную сессию и прокси.
2.  **Эмуляция активности**: Перед переходом в целевой канал бот просматривает список своих чатов, создавая видимость естественного открытия приложения.
3.  **Чтение контента**: Бот прокручивает ленту постов, делая паузы между блоками сообщений (чанками).
4.  **Регистрация просмотра**: Используется официальный метод API `GetMessagesViews`, который гарантирует зачет "глазика" на серверах Telegram.

---

## ⚙️ Параметры запуска

При запуске модуля вам необходимо указать:

*   **Ссылка на канал**: Публичный юзернейм (например, `@durov`) или ссылка.
*   **Лимит постов**: Количество последних сообщений в канале, которые нужно "прочитать".
*   **Параллельный режим**: Если включен, все выбранные аккаунты начнут работу одновременно.

---

## ⚡ Продвинутые функции

### Staggered Start (Разнесенный старт)
Для предотвращения детекции массовой активности, каждый аккаунт выбирает **случайную задержку** перед входом в сеть. Это позволяет имитировать органический рост просмотров в течение дня.

### Бесконечные ротации
После того как аккаунт просмотрел все указанные посты, он не завершает работу, а уходит в "сон" на несколько часов (настраиваемый интервал). После пробуждения цикл повторяется, что обеспечивает постоянную поддержку активности на канале.

---

## 💡 Рекомендации по безопасности

1.  **Индивидуальные прокси**: Крайне важно, чтобы каждый аккаунт имел свой прокси. Использование одного IP для 50 аккаунтов приведет к быстрой блокировке или обнулению просмотров.
2.  **Подписка**: Просмотры от участников канала ценятся Telegram выше. Рекомендуется сначала подписать аккаунты на целевой канал.
3.  **Рандомизация**: Система автоматически перемешивает порядок просмотра постов для каждого аккаунта, чтобы паттерны поведения не совпадали.
