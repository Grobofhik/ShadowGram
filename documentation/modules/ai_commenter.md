# <img src="../../resources/icons/moduls_icon.png" width="32" valign="middle"> AI Smart Commenter

Этот модуль позволяет вашим аккаунтам писать осмысленные, уникальные комментарии к постам в Telegram каналах, используя искусственный интеллект (AI).

Каждому аккаунту можно задать свой **собственный характер (промпт)**, благодаря чему комментарии с разных аккаунтов под одним постом будут выглядеть как живая дискуссия абсолютно разных людей.

---

## 🔑 Где взять бесплатный API ключ
Модуль работает по универсальному стандарту OpenAI. Мы рекомендуем использовать бесплатный и быстрый сервис **Groq**:
1. Перейдите на сайт [console.groq.com](https://console.groq.com/)
2. Авторизуйтесь через Google или GitHub.
3. Перейдите в раздел **API Keys** и нажмите **Create API Key**.
4. Скопируйте ключ (он начинается с `gsk_...`).

---

## ⚙️ Настройки модуля

*   **Ссылка на канал** — ссылка на целевой канал (например, `@durov`).
*   **Ваш API ключ** — ключ, который вы получили в Groq (или другом сервисе).
*   **Base URL API** — адрес сервера нейросети.
    *   Для Groq впишите: `https://api.groq.com/openai/v1`
    *   Для OpenRouter впишите: `https://openrouter.ai/api/v1`
*   **Название модели** — модель нейросети, которая будет генерировать текст.
    *   Для Groq рекомендуем: `llama3-8b-8192` или `gemma2-9b-it` (они очень быстрые).
*   **Сколько последних постов обработать** — сколько постов с конца канала скрипт будет пытаться комментировать.
*   **Вероятность оставить комментарий** — шанс (от 0 до 100), с которым бот решит написать ответ под постом.

## 🧠 Настройка индивидуальных промптов
Чтобы комментарии были разными:
1. Перейдите на главный экран ShadowGram (Список аккаунтов).
2. Нажмите на новую кнопку с иконкой ракеты 🚀 (AI Промпт) рядом с нужным аккаунтом.
3. Введите характер для этого профиля.

**Примеры промптов:**
*   *Аккаунт 1:* `Ты 20-летняя студентка. Напиши восторженный и короткий коммент на этот пост. Используй эмодзи.`
*   *Аккаунт 2:* `Ты суровый 40-летний инженер. Напиши скептический и короткий комментарий на этот пост. Без смайлов, пиши с маленькой буквы.`
*   *Аккаунт 3:* `Ты интернет-тролль. Напиши смешную короткую шутку по теме этого поста.`

Если промпт не задан, будет использован стандартный (просто позитивный короткий комментарий).

---

## 🛡️ Особенности работы
*   Скрипт читает посты и отправляет их в нейросеть. Длинные посты обрезаются до 1000 символов для экономии ваших лимитов.
*   Если в посте только картинка или видео без текста, нейросеть получит пометку `[Фото/Видео без текста]` и напишет комментарий исходя из этого факта.
*   Предусмотрены большие человечные паузы (10-30 секунд) после каждой успешной отправки комментария, чтобы не вызывать подозрений у антиспам-фильтров Telegram.
