# <img src="../../resources/icons/start_icon.png" width="32" valign="middle"> Smart Warmer (Умный прогрев)

Модуль **Smart Warmer** предназначен для имитации активности реального пользователя в Telegram. Это повышает уровень доверия (trust) со стороны мессенджера и снижает риск блокировки аккаунтов при выполнении массовых рассылок.

---

## 🔥 Что делает этот модуль?

Бот выполняет серию действий, которые характерны для обычного человека:

1.  **Вступление в чаты**: Переходит по предоставленным ссылкам и подписывается на группы.
2.  **Эмуляция чтения**: "Прокручивает" историю последних сообщений в чатах, делая короткие паузы.
3.  **Взаимодействие с контентом**: С определенным шансом (по умолчанию 20%) выбирает случайный пост и пересылает его в свои "Избранные сообщения" (Saved Messages).
4.  **Умные задержки**: После каждого действия бот делает паузу, длительность которой выбирается случайно в заданном вами диапазоне.

---

## ⚙️ Параметры модуля

*   **Ссылки на чаты**: Список ссылок на публичные или приватные группы (каждая ссылка с новой строки).
*   **Мин. задержка**: Нижний порог ожидания между действиями (в секундах).
*   **Макс. задержка**: Верхний порог ожидания.

---

## ⚡ Почему это важно для "фермы"?

Для Telegram новые или неактивные аккаунты, которые внезапно начинают слать сотни сообщений, выглядят подозрительно. 
**Smart Warmer** помогает:
*   Набрать историю действий (логов) на серверах Telegram.
*   "Разбавить" рекламную активность естественным поведением.
*   Увеличить срок жизни сессии перед выполнением основных задач.

---

## 💡 Советы по прогреву

*   **Постепенность**: Начинайте с вступления в 2–3 "белых" (без спама) чата в день.
*   **Разнообразие**: Используйте разные списки ссылок для разных аккаунтов.
*   **Интервалы**: Рекомендуется запускать прогрев раз в 2–3 дня в течение недели перед использованием аккаунта для рассылок.
