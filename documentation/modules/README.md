# <img src="../../resources/icons/moduls_icon.png" width="32" valign="middle"> Модули автоматизации

В этом разделе собраны подробные инструкции по работе с плагинами автоматизации ShadowGram. Каждый модуль предназначен для решения конкретной задачи и оптимизирован для работы с MTProto.

---

## 🚀 Список доступных инструментов

### 👁️ [Channel Viewer](channel_viewer.md)
Параллельная накрутка просмотров на посты Telegram-каналов. Поддерживает эмуляцию чтения, случайные задержки и ротации.

### 🔥 [Smart Warmer](smart_warmer.md)
Прогрев аккаунтов через имитацию реальной активности: вступление в группы, чтение сообщений, пересылка постов в "Избранное".

### 🛡️ [Privacy Guard](privacy_guard.md)
Мгновенная настройка параметров приватности аккаунта для защиты от спама и нежелательных звонков.

### 🖼️ [Set Avatar](set_avatar.md)
Быстрая установка или смена фотографии профиля для выбранного аккаунта.

### 🔍 [Check Session](check_session.md)
Глубокая диагностика сессий: проверка на бан, валидность ключей и автоматическая загрузка текущей аватарки.

### 🔑 [Get Auth Code](get_auth_code.md)
Перехват системных кодов авторизации от Telegram (777000) для входа в аккаунт на новых устройствах.

---

## ⚡ Особенности выполнения

*   **Параллельность**: Многие модули поддерживают запуск на всех выбранных аккаунтах одновременно.
*   **Логирование**: Каждое действие записывается в консоль логов в реальном времени.
*   **Безопасность**: Автоматическое использование индивидуальных прокси и случайных задержек.
