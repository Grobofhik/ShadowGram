# <img src="../../resources/icons/succure_icon.png" width="32" valign="middle"> Privacy Guard (Защита приватности)

Модуль **Privacy Guard** предназначен для моментальной настройки параметров безопасности ваших Telegram-аккаунтов. Это критически важный инструмент для защиты сессий от нежелательного внимания и спама.

---

## 🛡️ Какие настройки применяются?

Модуль устанавливает максимально строгие параметры конфиденциальности в один клик:

1.  **Номер телефона**: Устанавливается режим `Nobody` (Никто). Ваш номер не будет виден даже тем, кто добавил вас в контакты.
2.  **Поиск по номеру**: Ограничивается значением `My Contacts`. Посторонние люди не смогут найти ваш аккаунт, зная только номер телефона.
3.  **Время последнего входа**: Устанавливается `Nobody`. Никто не будет видеть, когда вы были в сети.
4.  **Звонки**: Разрешаются только от `My Contacts`. Это полностью блокирует входящий спам-звонки от ботов.
5.  **Приглашения в группы**: Запрещается добавление в чаты всем, кроме ваших контактов.

---

## ⚙️ Технические детали

*   **API Методы**: Модуль использует нативные методы Telegram API для управления приватностью (`account.setPrivacy`), что гарантирует моментальное применение настроек без задержек.
*   **Логирование**: После выполнения каждой операции бот выводит подтверждение в консоль.
*   **Изоляция**: Настройки применяются индивидуально для каждого выбранного аккаунта через его прокси-канал.

---

## 💡 Почему это важно?

Для работы с большим количеством аккаунтов "чистота" профиля — залог его долгожительства. 
*   **Защита от "угона"**: Скрытый номер телефона усложняет поиск аккаунта злоумышленниками.
*   **Снижение нагрузки**: Блокировка звонков и приглашений в группы освобождает аккаунт от лишнего "шума", который может привлечь внимание антифрод-систем.
