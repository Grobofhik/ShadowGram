# <img src="../../resources/icons/server_icon.png" width="32" valign="middle"> ServerGram: Удаленное управление

**ServerGram** — это высокопроизводительный backend-сервер на базе **FastAPI**, который работает в тесной связке с ShadowGram. Он позволяет перенести выполнение задач автоматизации с вашего компьютера на удаленный сервер (VPS) , что критически важно для работы 24/7 и управления большими фермами аккаунтов.

---

## 🛠️ Подготовка системы (Arch Linux)

Проект оптимизирован для работы на **Arch Linux**. Для стабильной работы ServerGram убедитесь, что в вашей системе установлены следующие компоненты:

1. **ОС**: Arch Linux.
2. **Python**: Версия 3.10 или выше.
3. **Gost**: Утилита для маршрутизации прокси.

### Установка системных зависимостей через pacman
Выполните следующие команды в терминале:

```bash
# Обновление системы и установка базовых инструментов
sudo pacman -Syu
sudo pacman -S python git base-devel

# Установка gost (доступен в официальных репозиториях или AUR)
sudo pacman -S gost
```

## 📦 Установка и запуск

### 1. Настройка окружения
Перейдите в директорию с серверной частью и создайте виртуальное окружение:
```bash
cd ServerGram
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Запуск сервера
Для запуска используйте `uvicorn`. Это обеспечит асинхронную обработку сотен соединений:
```bash
# Запуск на порту 8000
uvicorn Servergram:app --host 0.0.0.0 --port 8000
```

> <img src="../../resources/icons/note_icon.png" width="18" valign="middle"> **Важно:** Для продакшен-режима на Arch Linux рекомендуется создать `systemd` юнит, чтобы сервер автоматически поднимался после перезагрузки.

## 📡 Интеграция with ShadowGram

Чтобы клиент "увидел" ваш сервер:
1.  Откройте ShadowGram и перейдите в раздел <img src="../../resources/icons/server_icon.png" width="18" valign="middle"> **Сервер**.
2.  Введите URL сервера (например, `http://ваш-ip:8000`).
3.  Нажмите кнопку <img src="../../resources/icons/ping_icon.png" width="18" valign="middle"> **Проверить соединение**.
4.  Используйте функцию <img src="../../resources/icons/save_icon.png" width="18" valign="middle"> **Экспорт для сервера**, чтобы мгновенно передать все выбранные сессии на VPS.

## 🌟 Возможности ServerGram

*   **⚡ Параллельное выполнение**: Сервер сам распределяет нагрузку и запускает плагины на сотнях аккаунтов одновременно.
*   **📡 Живые логи**: Благодаря технологии **WebSockets**, вы видите всё, что происходит на сервере, прямо в консоли вашего клиента.
*   **⚖️ Flood-защита**: Встроенные семафоры ограничивают количество активных коннектов к Telegram, предотвращая "вылеты" сессий.
*   **🔄 Общие плагины**: Любой плагин, написанный вами для ShadowGram, копируется на сервер и работает там без изменений.
